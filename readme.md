# Journal telegram bot
Bot was created to analyze where time goes.

## Available features:
- Set time marks of starting an activity

## How to run
1. Create `.env` file and write this the bot token:
```
BOT_TOKEN={your_bot_token}
```
2. Run `npm run start`
